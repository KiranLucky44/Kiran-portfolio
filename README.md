# Portfolio (React + Vite + TypeScript)

This is a minimal starter scaffold for a personal portfolio built with React, Vite and TypeScript.

Getting started (PowerShell / Windows):

```powershell
cd "C:\Users\kiran\OneDrive\Desktop\portfolio"
npm install
npm run dev
```

Files created:
- `index.html` - Vite entry
- `vite.config.ts` - Vite + React plugin
- `tsconfig.json` - TypeScript config
- `src/main.tsx` - React entry
- `src/App.tsx` - Minimal app
- `src/index.css` - Basic styles

Next steps:
- Install additional tools (Tailwind, ESLint rules) if desired.
- Replace placeholder content in `src/App.tsx` with your portfolio content.
